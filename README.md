# Financial Helper — Личный финансовый менеджер

Это консольное Java-приложение для управления личными финансами, разработанное в рамках курса ООП.  
Поддерживает нескольких пользователей, авторизацию, добавление доходов/расходов по категориям, просмотр статистики и сохранение данных в файл.

### Основные возможности
- Регистрация и авторизация пользователей
- Добавление доходов и расходов с категориями и описаниями
- Просмотр текущего баланса и статистики по категориям
- Список всех операций с описаниями
- Данные сохраняются в файл `users_data.txt` и загружаются при запуске
- Полностью в памяти (без базы данных)
